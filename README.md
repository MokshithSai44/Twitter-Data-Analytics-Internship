# Twitter Analytics Dashboard

## Overview
Interactive Power BI dashboard analyzing tweet engagement, impressions, retweets, likes, and interaction trends.

## Features
- KPI Cards
- Engagement Analysis
- Media Interaction Trends
- Top Performing Tweets
- Interactive Filters & Slicers
- Conditional Visual Logic

## Tools Used
- Power BI
- DAX
- Data Cleaning
- Data Visualization

## Filters Applied
- Even impressions only
- Odd tweet dates
- Word count restrictions
- Weekend exclusion
- Time-based visibility logic# Twitter-Data-Analytics-Internship
